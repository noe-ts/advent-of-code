# Advent of Code - TypeScript

Solutions for Advent of Code problems using TypeScript, Bun, and Vitest.

## Setup

1. Install dependencies:
```bash
bun install
```

## Usage

### Running Solutions

Run a specific day's solution:
```bash
bun run src/2024/day01/index.ts
```

### Testing

Run all tests:
```bash
bun test
```

Run tests in watch mode:
```bash
bun test:watch
```

Run tests with UI:
```bash
bun test:ui
```

## Structure

```
src/
  YYYY/
    day-XX/
      index.ts      # Solution implementation
      index.test.ts # Tests
      input.txt     # Puzzle input (optional)
```
